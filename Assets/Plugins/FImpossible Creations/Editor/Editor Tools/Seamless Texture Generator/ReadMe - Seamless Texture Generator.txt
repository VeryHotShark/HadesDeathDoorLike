To use Seamless Texture Generator click on any texture file in your project
with right mouse button, then go to "FImpossible Creations (probably at the bottom of list) > Textures > Seamless Generator Window"
Window of "Seamless Texture Generator" will pop up and load selected file to operate on.
When you tweak your settings, click "Apply to file" then you will have seamless texture.
(Supported formats .jpg, .png, .exr, .tga)

Description:

Seamless Texture Generator is simple but handy tool to use within Unity Editor.
If you often fight with making textures seamless by hand or sending them back and forth 
to your graphic designers, then you can forget about struggling with that.

Seamless Texture Generator will loop your textures so you can tile them and it 
will be much less noticeable. Package is providing pleasant custom inspector 
window with clean preview of what will happen with your full texture. 
You can customize settings to create best fitting set for your texture.

Package's Algorithm is supporting image file formats like .jpg .png .exr and .tga
Later package will be developed more to give more advanced options for 
texture tiling so you will be able to create better seamless textures!
Algorithm is supporting making seamless textures also for normal maps!

MAIN FEATURES:
• Generating seamless textures from files inside project
• Functional custom inspector window
• Clean preview for seamless generation process
• More functions soon when package will be developed more
• Full source code included
